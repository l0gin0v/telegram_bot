# telegram_bot
Task 1:
  - реализовать консольного бота, который умеет задавать вопросы пользователю и проверять правильность пользовательских ответов.

Task 2:
  - перенести существующий функционал на API Telegram, заменив консоль на чат телеграма в качестве среды для взаимодействия с пользователем.

Task 3:
  - реализовать базовый функционал бота для погоды. Пользователь выбирает свой город/посёлок и бот предоставляет ему выбор, на какой день показать данные о погоде (сегодня, завтра, послезавтра, на 7 дней). Выбор дня осуществляется через кнопки телеграма.

Task 4:
  - Реализовать уведомления. Пользователь выбирает время ежедневной рассылки. Каждый день, в выбранное время, присылается уведомление с погодой, в выбранном городе.
